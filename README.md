# QA Login Testing Project

This repository demonstrates my QA skills using a simple **Login functionality**
as a test target.  
The project focuses on **manual testing fundamentals** and clear documentation
following real-world QA practices.

---

##  Scope of Testing
The following areas were covered:
- Functional testing of Login feature
- Positive and negative scenarios
- Input validation
- Error handling
- UI and localization issues
